CareBeat Monitor
Smart. Simple. Life-saving.
A wearable infant health system that tracks respiration and heart rate through Bluetooth and Python.

1. Overview
CareBeat Monitor is a low-cost wearable that helps parents monitor an infant’s breathing in real time. It uses an Adafruit BLE sensor and a custom Python desktop app to display live data, calculate respiration rate, and record sessions.

2. Why It Matters
Every breath counts. This system was built to give parents peace of mind through accessible health monitoring.
